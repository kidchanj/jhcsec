Privacy Policy — E4sy_Proxy

Effective date: 2025-09-02

1. Purpose

This Chrome extension is designed solely to let users switch proxy connection modes within the browser (Direct, Proxy, System, or domain-specific proxy) quickly and locally.

2. What Data We Collect

We do not collect or transmit any personally identifiable information (PII), browsing history, authentication data, payment information, or analytics.

The only data stored is the proxy server host, port, and the last selected mode and domain choice. These values are saved locally in your browser via Chrome Storage and are never sent to external servers.

3. Why We Request Permissions

  • proxy: to switch Chrome’s network routing between Direct/Proxy/System/PAC modes.
  • storage: to persist your proxy host/port and last-selected mode on the local device.
  • tabs: to read the active tab’s URL in order to compute the registrable top-level domain for domain-specific proxy rules.
  • host_permissions (<all_urls>): required to accurately access the active tab’s domain. It is used solely for domain calculation and never to read or modify page content.

4. Remote Code

No remote code is fetched or executed. All logic is packaged with the extension.

5. Data Sharing

We do not sell, share, or transfer user data to third parties. No data is transmitted to external servers as part of the extension’s operation.

6. Children’s Privacy

This extension targets general audiences and does not knowingly process information from children.

7. Changes to This Policy

If the policy changes, we will update the effective date at the top of this page. Feature additions—if any—will remain limited to what is necessary to operate proxy switching.

8. Contact

If you have questions, contact the developer at hyechan0505@google.com

개인정보 처리방침 — E4sy_Proxy

시행일: 2025-09-02

1. 목적

이 확장 프로그램은 브라우저 내에서 프록시 연결 방식을 (Direct, Proxy, System, 도메인별 프록시) 빠르고 로컬로 전환하는 것만을 목적으로 합니다.

2. 수집하는 데이터

개인식별정보(PII), 방문 기록, 인증 정보, 결제 정보, 분석 정보 등은 수집하거나 전송하지 않습니다.

저장되는 정보는 프록시 서버 주소(host), 포트(port), 마지막으로 선택한 모드 및 도메인 선택 값뿐이며, 이는 브라우저의 Chrome Storage에 로컬로 저장되며 외부 서버로 전송되지 않습니다.

3. 권한 사용 근거

  • proxy: Direct/Proxy/System/PAC 모드 간 네트워크 라우팅을 전환하기 위해 필요합니다.
  • storage: 프록시 호스트/포트 및 마지막 선택 모드를 로컬에 저장하기 위해 필요합니다.
  • tabs: 활성 탭의 URL을 확인하여 도메인별 프록시 적용을 위한 최상위 등록 가능 도메인(eTLD+1)을 계산하기 위해 필요합니다.
  • host_permissions (<all_urls>): 활성 탭의 도메인을 정확히 확인하기 위해 필요합니다. 도메인 계산 이외의 목적으로 페이지 내용을 읽거나 수정하지 않습니다.

4. 원격 코드

원격 서버에서 코드를 받아 실행하지 않습니다. 모든 로직은 확장 프로그램 패키지 내에 포함되어 있습니다.

5. 데이터 공유

사용자 데이터를 제3자에게 판매·공유·전송하지 않습니다. 확장 프로그램 동작 과정에서 외부 서버로 데이터가 전송되지 않습니다.

6. 아동의 개인정보

이 확장 프로그램은 일반 사용자를 대상으로 하며, 아동의 정보를 알 knowingly 수집·처리하지 않습니다.

7. 정책 변경

정책이 변경되면 본 문서 상단의 시행일을 갱신합니다. 기능 추가가 있더라도 프록시 전환에 필요한 범위를 벗어나지 않습니다.
